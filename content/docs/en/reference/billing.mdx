title: Billing and LLM Capacity
description: Understand how plans differ by included LLM API capacity.

Plans do not gate product features.

Every plan includes the same desktop app, agent preset repository, Blueprint Builder, local execution, runtime orchestration, and Observation capabilities.

The difference between tiers is included LLM API usage capacity.
