title: Agent Presets
description: Reusable agent roles with prompts, tools, MCP servers, profiles, and skills.
---

An agent preset is a durable role definition.

It can include:

- Identity and role instructions.
- Model configuration.
- Mounted profiles.
- Allowed tools.
- MCP servers.
- Skills.

## Why presets matter

Presets keep multi-agent systems composable. You should reuse a tested Architect, Developer, Verifier, or Manager instead of rewriting prompts for every workflow.

## Good preset design

A strong preset has a narrow responsibility, clear authority, and a small tool surface.
