title: Agent
description: Create Agent presets with independent models, prompts, and tools.
---

In The One's Multi-Agent system, every Agent has independent configuration: model, prompt, and tools. That is why different Agents can take sharply different roles.

Start by creating your first Agent:

1. Give the Agent a clear name.
2. Choose the model it should use.
3. Select Prompt / Skill / MCP modules and optional supplemental Prompt.
4. Select built-in tools, which determine the Agent's orchestration capability inside the Graph and its ability to interact with human users.
