The Zeroth is the platform. The One is the desktop app that runs on your machine.

Use these docs when you want to design agent presets, wire blueprints, launch runs in local workspaces, and inspect what every agent is doing.

## Recommended path

1. Install The One.
2. Sign in and authorize the desktop app.
3. Create or open a blueprint.
4. Launch the workflow from a local workspace.
5. Observe messages, context, tool calls, and handoffs while the run is alive.

## What to read first

- Install The One if you are setting up a new machine.
- First Blueprint if you want the fastest usable workflow.
- Agent Presets if you are defining reusable agent roles.
- Runtime Orchestration if you are building workflows that change shape while running.
