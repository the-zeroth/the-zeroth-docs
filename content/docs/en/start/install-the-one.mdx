---
title: Install The One
description: Install the desktop app for The Zeroth and connect it to your account.
---

The One is the desktop app for The Zeroth. Install it on the machine where your target workspaces live.

## Steps

1. Download the latest Windows installer from the Download page.
2. Run the installer.
3. Open The One.
4. Sign in with your The Zeroth account.
5. Complete desktop authorization from the Console.

## What authorization does

Authorization binds the desktop app to your account. It does not require you to paste provider API keys into the desktop app.

## After install

Open a local workspace and launch a blueprint from The One.
