---
title: First Blueprint
description: Create a small blueprint and run it from a local workspace.
---

A blueprint is an executable graph of agents. Nodes bind to agent presets. Edges define how work moves.

## Build a minimal blueprint

1. Create two agent presets: Planner and Implementer.
2. Add two nodes to the blueprint.
3. Bind each node to one preset.
4. Connect Planner to Implementer.
5. Save the blueprint.

## Run it

Choose a local workspace, start the blueprint, and watch the run from Observation.

## Keep it small

Your first blueprint should prove the routing model. Do not start with five agents and ten branches.
