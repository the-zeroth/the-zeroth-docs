---
title: 安装 The One
description: 安装 The Zeroth 的桌面端应用，并绑定到你的账户。
---

The One 是 The Zeroth 的桌面端应用。把它安装在你的目标工作区所在的电脑上。

## 步骤

1. 从 Download 页面下载最新 Windows 安装包。
2. 运行安装包。
3. 打开 The One。
4. 使用 The Zeroth 账户登录。
5. 在 Console 中完成桌面端授权。

## 授权做了什么

授权会把 The One 绑定到你的账户。你不需要手动把模型供应商 API key 粘贴到桌面端。

## 下一步

打开一个本地工作区，并从 The One 启动一张 Blueprint。
