---
title: 快速入门
description: 从安装 The One 到运行第一个 Multi-Agent Graph 的最短路径。
---

本指南将带你从安装开始，到在 The One 中运行一个你专属的 Multi-Agent。

1. 安装 The One 并登录

前往官网下载 The One 安装包。

2. 从官方默认的仓储 Space，或从在分享平台上下载你喜欢的 Space 开始

![界面截图](/docs-assets/images/image-20260531103928-b9rodqe.png)

3. 导入你常用的 Prompt，Skill 和 MCP，用你喜欢的方式来调整 Agent 和 Graph

![界面截图](/docs-assets/images/image-20260531104008-xjbdzxl.png)

4. 选择一个 Graph 开始运行，选择其中担任入口角色的 Agent（如 PM）开始对话，就像你在其他 AI Agent 桌面端应用中做的那样

![界面截图](/docs-assets/images/image-20260531104121-uinbxt8.png)
