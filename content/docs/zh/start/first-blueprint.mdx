---
title: 第一张 Blueprint
description: 创建一张小型 Blueprint，并从本地工作区运行它。
---

Blueprint 是一张可执行的 Agent 图。Node 绑定到 Agent 预设，Edge 定义任务如何流转。

## 创建最小 Blueprint

1. 创建两个 Agent 预设：Planner 和 Implementer。
2. 在 Blueprint 中添加两个 node。
3. 把每个 node 绑定到一个预设。
4. 连接 Planner 到 Implementer。
5. 保存 Blueprint。

## 运行

选择一个本地工作区，启动 Blueprint，然后在 Observation 中观察运行过程。

## 保持小

第一张 Blueprint 只需要验证路由模型。不要一上来就做五个 Agent 和十条分支。
