The Zeroth 是平台。The One 是运行在你本机上的桌面端应用。

当你要设计 Agent 预设、搭建 Blueprint、从本地工作区启动运行，并观察每个 Agent 的工作过程时，从这里开始。

## 推荐路径

1. 安装 The One。
2. 登录并授权桌面端。
3. 创建或打开一张 Blueprint。
4. 从本地工作区启动工作流。
5. 在运行期间观察消息、上下文、工具调用和 handoff。

## 先读什么

- 新机器安装：读 Install The One。
- 想最快跑起来：读 First Blueprint。
- 要定义复用角色：读 Agent Presets。
- 要构建运行时会变化的工作流：读 Runtime Orchestration。
