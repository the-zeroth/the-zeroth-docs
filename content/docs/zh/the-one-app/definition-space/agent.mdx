---
title: Agent
description: 创建具有独立模型、提示词和工具能力的 Agent 预设。
---

在我们的 Multi-Agent 中，每一个 Agent 的配置都是彼此独立的，包括模型，提示词和工具；正是因此我们才能让每个 Agent 担任截然不同的角色

首先从创建第一个 Agent 开始，

1. 为你的 Agent 起个好听的名称

2. 选择要使用的模型

3. 选择要插入的 Prompt / Skill / MCP 模块，以及可选的补充性 Prompt

4. 选择要使用的内置 Tool（关系到模型在 Graph 中的编排能力以及与人类用户交互的能力）
