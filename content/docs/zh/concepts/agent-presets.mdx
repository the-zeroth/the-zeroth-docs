title: Agent 预设

Agent 预设是一种稳定的角色定义。

它可以包含：

- 身份和角色指令。
- 模型配置。
- 挂载的 profile。
- 允许使用的工具。
- MCP server。
- Skill。

## 为什么预设重要

预设让多智能体系统可以组合。你应该复用验证过的 Architect、Developer、Verifier 或 Manager，而不是每个 workflow 都重新写 prompt。

## 好的预设设计

好的预设应该职责窄、权限清晰、工具面小。
