# The Zeroth Docs

Public documentation source for [The Zeroth](https://the-zeroth.com).

The Zeroth is a desktop platform for designing, running, and observing real multi-agent workflows. Its desktop app, The One, lets users create reusable agent presets, compose them into visual blueprints, execute workflows in local workspaces, and inspect agent communication, context, and tool calls while runs are active.

This repository contains the public documentation content used by the official website. It is intentionally content-only and does not include product source code or deployment configuration.

## Content

Documentation lives in `content/docs`:

- `en/` contains English docs.
- `zh/` contains Chinese docs.
- `meta.json` files control page ordering and navigation groups.

## Tech Stack

- MDX for documentation pages.
- Fumadocs-compatible `meta.json` files for navigation structure.
- Locale-first directory layout for multilingual content.
