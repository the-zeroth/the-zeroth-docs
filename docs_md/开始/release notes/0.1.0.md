The One——一款专门为Multi-Agents（多智能体）架构的开发，探索，应用所设计的开创性桌面应用

我们都知道，多智能体是 AI 智能体的未来，但人们仍在苦苦挣扎——耗费巨大精力在重复的代码实现上，因此只能艰难地探索着仅有的几种简单多智能体架构。

因此这款软件的理念便是，用一款软件来搭建任意人们能够想象的Multi-Agents（多智能体）架构。

‍

程序分为两大部分:  
1.仓储界面，包括：  
1）可插拔的prompt/skill模块

2）可插拔的tool/mcp模块

3）自定义组装的agent预设

4）将一个个不同的agent通过任意的边（通讯权限）连接在一起所组成的Multi-Agents Graph蓝图

2.Graph运行界面特点：

1）可靠：就像几款热门的本地桌面端agent应用那样，可以在任意指定路径下让agent按照你的指示开始工作，不同点在于我们运行的是我们自己创建的Multi-Agents

2）透明：不同于其他agent软件倾向于黑箱式的运行方式，我们的软件追求的是agent运行过程绝对的透明，这样才能方便用户探索不同的agent架构，在调试过程中找出问题所在；另外，由于我们在设计之初就是出于multi-agent设计的，因此我们有针对性的独特的可视化设计——以上帝视角实时浏览各个 Agent 之间的沟通、思考与协作过程，以及每个agent自身详细的上下文组成和工具调用信息

3）动态：基于我们专为Multi-Agents架构设计的强大内置tool，agents可以在运行过程中自行拓展Graph的结构
