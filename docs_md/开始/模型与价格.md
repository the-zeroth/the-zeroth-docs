# The One 支持的模型渠道与收费方式

The One 是 The Zeroth 的 AI Agent 桌面端应用。你可以直接使用 The Zeroth 官方模型服务，也可以接入你自己已有的第三方模型账号。

我们支持两种使用方式：

1. 使用 The Zeroth 官方模型服务
2. 接入你自己的第三方模型渠道

## 一、官方模型服务

这是最简单、最推荐的方式。

登录 The Zeroth 账户后，你可以直接在 The One 中使用官方模型服务，不需要自己准备 API Key，也不需要单独配置模型供应商。

The Zeroth 官方模型服务基于 OpenRouter 提供模型访问能力。选择官方模型服务有两个核心优势：

### 1. 模型选择更完整，服务更稳定

OpenRouter 聚合了大量主流和前沿模型，你可以在 The One 中使用更丰富的模型选择，包括 OpenAI、Anthropic、Google、Meta、Mistral、DeepSeek 等不同模型生态中的能力。

相比单独接入某一家模型供应商，官方模型服务可以让你更灵活地为不同 Agent、不同任务选择合适模型。

同时，官方服务由 The Zeroth 统一接入和维护，减少你自己管理多个供应商 API 的复杂度。

### 2. 官方账户价格更优惠

The Zeroth 通过官方账户统一接入模型服务，整体价格通常比个人直接开通和使用更有优势。

这意味着你可以用更低的使用门槛开始使用高质量模型，不需要自己分别注册多个模型平台、充值、配置和维护 API Key。

官方模型服务适合：

|使用场景|是否推荐|
| ----------------------------| ----------|
|第一次使用 The One|推荐|
|不想配置 API Key|推荐|
|希望模型账单统一管理|推荐|
|想快速体验多种模型|推荐|
|希望减少供应商账号管理成本|推荐|

## 二、第三方模型渠道

如果你已经有自己的模型供应商账号，也可以在 The One 中添加自己的 API Key 或完成 OAuth 授权。

目前支持的第三方渠道包括：

|渠道|支持方式|
| ------------------| ------------------------------------|
|OpenAI|API Key|
|Anthropic Claude|API Key / Claude Code OAuth|
|Google Gemini|API Key / Gemini Code Assist OAuth|
|OpenAI Codex|OAuth|
|自定义兼容服务|API Key|

自定义兼容服务适合已经有企业模型网关、私有部署模型、或兼容 OpenAI 接口的第三方服务的团队。

使用第三方渠道时，模型调用不会消耗 The Zeroth 官方模型服务额度；费用、额度、速率限制和可用模型范围由对应第三方平台决定。

## 三、收费方式

The Zeroth 官方模型服务支持两种收费方式：

1. 订阅制
2. 用量包制

### 订阅制

订阅制适合持续使用 The One 的用户。不同订阅档位包含不同的官方模型服务使用额度。

当前订阅档位包括：

|套餐|价格|适合用户|
| ---------| -----------| -------------------------------------------|
|Starter|$20 / 月|轻量使用、个人体验、低频 Agent 工作流|
|Pro|$60 / 月|日常稳定使用、多 Agent 工作流、较高频任务|
|Max|$200 / 月|高频使用、复杂工作流、重度自动化场景|

所有订阅档位都可以使用 The One 的核心能力，包括：

- 桌面端应用
- Agent 预设
- Blueprint 工作流
- 本地执行
- 运行观察
- 官方模型服务
- 第三方模型渠道接入

套餐之间的主要区别是官方模型服务的可用额度和适合的使用强度不同。

### 用量包制

如果你当前订阅中的官方模型服务额度不够，可以额外购买用量包。

用量包适合：

- 某个月临时使用量变高
- 不想升级长期订阅
- 某次项目、交付或集中测试需要更多模型额度

当前用量包为：

|用量包|价格|说明|
| --------| ----------| --------------------------------|
|Top-up|$20 / 次|为官方模型服务额外增加可用额度|

用量包不会改变你的订阅套餐，只会增加你的官方模型服务可用额度。

## 四、第三方渠道如何收费

如果你使用自己的 OpenAI、Anthropic、Gemini 或自定义模型服务，相关费用由对应第三方平台收取。

例如：

|使用方式|费用归属|
| ------------------------------------| ------------------------------------------|
|使用 The Zeroth 官方模型服务|消耗 The Zeroth 账户中的官方模型服务额度|
|使用自己的 OpenAI API Key|由 OpenAI 结算|
|使用自己的 Anthropic API Key|由 Anthropic 结算|
|使用自己的 Gemini API Key 或 OAuth|由 Google 或对应账号权益决定|
|使用企业内部模型网关|由你的团队或企业内部规则决定|

The Zeroth 不会对第三方模型调用重复扣除官方模型服务额度。

## 五、怎么选择

如果你只是想开始使用 AI Agent，推荐选择官方模型服务。它配置最少，模型选择丰富，价格更友好，也更容易统一管理。

如果你已经有自己的供应商账号，或者公司要求模型调用必须走内部账号，可以选择第三方模型渠道。

|使用场景|推荐方式|
| --------------------------------------| ----------------|
|不想配置 API Key|官方模型服务|
|想快速使用多种模型|官方模型服务|
|希望使用更优惠的官方账户价格|官方模型服务|
|已经有 OpenAI / Claude / Gemini 账号|第三方渠道|
|公司有统一模型网关|自定义兼容服务|
|想把模型费用走自己的供应商账单|第三方渠道|

## 总结

The One 支持两种模型使用模式：

- 官方模型服务：基于 OpenRouter，模型选择丰富、服务可靠、价格更优惠，使用 The Zeroth 的订阅额度或用量包额度。
- 第三方模型渠道：连接你自己的模型账号，费用由第三方平台或你的团队结算。

对大多数用户，官方模型服务是最省心的选择。  
对已经有模型基础设施的团队，第三方渠道可以满足更灵活的接入需求。
